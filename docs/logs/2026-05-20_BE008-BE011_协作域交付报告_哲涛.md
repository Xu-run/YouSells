## 7. 环境差异说明

原项目配置预期 Docker 环境（MySQL:13306, Redis:16379）。本次交付同步做了以下调整：

| 项目 | 原始值 | 调整 | 原因 |
|------|--------|------|------|
| Redis 依赖 | `spring-boot-starter-data-redis` | **已移除** | 项目未使用 Redis，且本地未安装 Redis |
| Redis 配置 | application*.yml 中 data.redis | **已移除** | 同上 |
| MySQL 端口 | 13306 (Docker 映射) | 通过 `MYSQL_PORT=3306` 环境变量覆盖 | 本地 MySQL 原生安装 |
| MySQL 字符集 | — | 导入 SQL 需指定 `--default-character-set=utf8mb4` | Windows MySQL 客户端默认 gbk |
| users 表 | 空表 | 补充 admin/member 两行 | 硬编码认证用户需与数据库对齐 |
