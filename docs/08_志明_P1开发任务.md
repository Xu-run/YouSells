## 3. BE-102 客户模型重建

- **状态**：🔴 未开始
- **前置任务**：BE-101（用户体系，已完成）
- **时间**：Week 2-3
- **涉及模块**：`modules/customer/**`

### API 契约

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/api/customers?page=&pageSize=&keyword=&grade=&major=&progress=&intent=&ownerUserId=` | 分页列表，多维筛选 |
| GET | `/api/customers/{id}` | 详情含邀约备注 |
| POST | `/api/customers` | 新建学生档案 |
| PUT | `/api/customers/{id}` | 更新客户 |

### 客户数据模型

| 字段 | 类型 | 说明 |
|---|---|---|
| `real_name` | VARCHAR(64) | 真实姓名 |
| `grade` | VARCHAR(16) | 年级（大一/大二/大三/大四） |
| `major` | VARCHAR(128) | 专业 |
| `class_name` | VARCHAR(64) | 班级 |
| `inviter_user_id` | BIGINT | 邀约人 ID |
| `owner_user_id` | BIGINT | 负责人 ID |
| `progress` | VARCHAR(16) | 进度：职规 / 技术栈 / 课程 |
| `intent` | VARCHAR(8) | 意向：很稳 / 可跟 / 观望 / 冷淡 |
| `inviter_note` | TEXT | 邀约备注（邀约人一次性填写） |

### 进度流转（可回退）

```
职规 ──→ 技术栈 ──→ 课程
  ↑         ↑        ↑
  └─────────┴────────┘
```

意向与进度独立：进度是"聊到哪了"，意向是"成功可能性"。

### 验收标准

- [ ] `GET /api/customers` 分页返回学生列表（姓名+年级+专业+班级+进度+意向+负责人）
- [ ] 支持按年级、专业、进度、意向、负责人多维筛选
- [ ] `GET /api/customers/{id}` 返回完整详情含邀约备注
- [ ] `POST /api/customers` 可新建学生档案

## 4. BE-103 跟进记录改造

- **状态**：🔴 未开始
- **前置任务**：BE-102（客户模型）
- **时间**：Week 3
- **涉及模块**：`modules/followup/**`

### API 契约

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/api/follow-ups?customerId=` | 某客户的所有跟进，按时间排序 |
| POST | `/api/follow-ups` | `{ customerId, userId, progress, content, feedback, nextAction }` |

### 跟进记录字段

| 字段 | 类型 | 说明 |
|---|---|---|
| `customer_id` | BIGINT | 关联客户 |

| `content` | TEXT | 沟通内容 |
| `feedback` | VARCHAR(500) | 学生反馈 |
| `next_action` | VARCHAR(255) | 下一步动作 |
| `created_at` | DATETIME | 时间戳 |

## 5. BE-106 攻略区

- **状态**：🔴 未开始
- **前置任务**：BE-101（独立于客户域，可与 BE-102 并行）
- **时间**：Week 3-4
- **涉及模块**：`modules/topic/**`（全新模块）

### API 契约

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/api/topics?category=&keyword=&page=&pageSize=` | 问题列表，分类筛选+搜索 |
| GET | `/api/topics/{id}` | 问题详情含回答列表 |
| POST | `/api/topics` | 新建提问 `{ title, description, category }` |
| POST | `/api/topics/{id}/replies` | 写回答 `{ content }` |
| PATCH | `/api/topics/{id}/solved` | 标记已解决 |
| PATCH | `/api/topics/{id}/replies/{replyId}/solution` | 标记最佳方案 |

### 数据模型

**topics（问题）**

| 字段 | 类型 | 说明 |
|---|---|---|
| `title` | VARCHAR(255) | 标题 |
| `description` | TEXT | 详细描述 |
| `category` | VARCHAR(16) | 分类：邀约 / 沟通 / 跟进 / 转化 / 其他 |
| `author_user_id` | BIGINT | 提问人 |
| `solved` | TINYINT(1) | 是否已解决 |
| `created_at` / `updated_at` | DATETIME | 时间戳 |

**topic_replies（回答）**

| 字段 | 类型 | 说明 |
|---|---|---|
| `topic_id` | BIGINT | 关联问题 |
| `user_id` | BIGINT | 回答人 |
| `content` | TEXT | 方案内容 |
| `is_solution` | TINYINT(1) | 是否被标记为最佳方案 |
| `created_at` | DATETIME | 时间戳 |
