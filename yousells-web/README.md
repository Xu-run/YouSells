# YouSells Web

## 运行前说明

当前前端技术栈：

- `Vue 3`
- `TypeScript`
- `Vite`
- `Pinia`
- `Vue Router`
- `Element Plus`

PowerShell 如果拦截了 `npm.ps1`，可以改用：

```powershell
cmd /c npm install
cmd /c npm run dev
```

## 当前骨架范围

- 登录页
- 主布局与路由守卫
- 首页看板
- 客户总表
- 客户详情与跟进记录
- 公共安排
- 日报 / 周报
- 话术库

这些页面已经接好了 API 文件与基础展示结构，后续可以直接继续做正式业务开发与联调。
